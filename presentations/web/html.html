If you're wondering: why is the text aligned so terribly?

this is called Indent Styling

Indent Styling is a convention used to make code

easily understandable

to human readers

Much like index styling, comments are another convention used to make code

easily understandable

to human readers

Comments are composed of three parts:

<!--

opening tag

comment

comment text

-->

closing tag

Take a look at a more complex container system with Comments:

<!-- START main container -->
<div>
    <!-- START navigation -->
    <div>
        <div>Button 1</div>
        <div>Button 2</div>
        <div>Button 3</div>
    </div>
    <!-- END navigation -->
    <!-- START main content -->
    <div>
        <div>
            <h1>I ♥ Headers</h1>
            <p>Lorem ipsum dolor sit amet!</p>
        </div>
    </div>
    <!-- END main content -->
</div>
<!-- END main container -->

Indent Styling and Comments won't affect your designs, as they

only help human readers

(devices ignore it all)
